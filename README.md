# Data Cleaning - Medical Appointment No Shows

## What I Did

In this project, I cleaned the dataset to make it ready for analysis. The following steps were performed:

- Loaded the CSV file using Pandas
- Checked and removed missing values
- Removed duplicate rows
- Converted text values (like gender) into a standard format
- Changed date columns into proper datetime format
- Renamed column names to lowercase and removed spaces
- Fixed data types (for example, age converted to integer)

## Result

After cleaning:
- The dataset has no missing or duplicate values
- All data is in a proper and consistent format
- The dataset is ready for further analysis

- Python
- Pandas
